COMMAND LINE JAR

DOWNLOAD: sokrates-0.9.jar (43 MB)

Usage: 

  java -jar sokrates-1.0.jar init [options]
       -confFile <arg>   [OPTIONAL] the path to configuration file (default is "<srcRoot>/_sokrates/config.json")
       -srcRoot <arg>    the path to source code root folder

  java -jar sokrates.jar generateReports [options]
         -confFile <arg>               [OPTIONAL] the path to configuration file (default is "<srcRoot>/_sokrates/config.json")
         -outputFolder <arg>           the folder where reports will be stored (default is "<srcRoot>/_sokrates/reports/")
         -reportAll                    generate all reports
         -reportControls               generate only the controls report (stored in <outputFolder>/Controls.html)

         -reportOverview               generate only the report describing the overview of files in scope
         -reportDuplication            generate only the duplication report (stored in <outputFolder>/Duplication.html)
         -reportLogicalDecomposition   generate only the components logical decomposition report (stored in
                                       <outputFolder>/LogicalDecomposition.html)
         -reportFileSize               generate only the file size report (stored in <outputFolder>/FileSize.html)
         -reportUnitSize               generate only the unit size report (stored in <outputFolder>/UnitSize.html)
         -reportCyclomaticComplexity   generate only the cyclomatic complexity report (stored in
                                       <outputFolder>/CyclomaticComplexity.html)
         -reportCrossCuttingConcerns   generate only the cross cutting concerns report (stored in
                                       <outputFolder>/CrossCuttingConcerns.html)
         -reportMetrics                generate only the metrics overview report (stored in <outputFolder>/Metrics.html)
         -reportJson                   save only report data in JSON format
         -reportText                   save only textual summary
