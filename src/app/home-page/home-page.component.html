WHAT IS SOKRATES?

  • Sokrates is a tool built by Željko Obrenović. It implements his vision on how to document and analyze software architectures of complex systems.
  • Sokrates is based on the following premises (borrowing last two from Grady Booch):
    • Any software architecture documentation that is not connected to the source code is doomed to become outdated and useless.
    • The best formal and unambiguous specification of a software-intensive system is its executable code.
    • Executable code as specification is, however, insufficient for visualizing, reasoning about, understanding, and communicating about that system.
  • Sokrates solves the problem of software architecture documentation by providing a pragmatic way to describe key elements of software architectures and connect them explicitelly to the source code.
  • Based on a formal description of different software architecure aspects, sokrates generates a number of reports and visualization that can help you to reason about, understand, and communicate about the software system.
  • Sokrates borrows ideas from code spelunking tool, in particular grep, adding structure on top of regex source code searches.
  • Sokrates comes with both command line interface and interactive GUI code explorer.
